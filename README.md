# kindle

### Run Server
```bash
cd epub-press/

docker-compose -f docker-compose.yaml up -d
```


### Run Python
```bash
python3 -m venv venv
source venv/bin/activate
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```
